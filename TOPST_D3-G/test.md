## 2. 글 형식 수정
- `**bold 굵게**` → **bold 굵게**
- `*italic 이텔릭*` → *italic 이텔릭*
- `~~strikethrough~~ ~~(시작)내용(끝)~~` → ~~strikethrough~~

## 3. 목록화

### 넘버링 없이
```
- Item 1
  - Subitem 1.1
  - Subitem 1.2
- Item 2
```

### 넘버링 존재
```
1. First item
2. Second item
   2.1. Subitem 2.1
   2.2. Subitem 2.2
3. Third item
```

## 6. 표 넣기
```markdown
| Header 1 | Header 2 | Header 3 |
|----------|----------|----------|
| Row 1    | Data     | Example  |
| Row 2    | More     | Data     |
```

### html 문법으로 표 넣기
---
<table>
  <thead>
    <tr>
      <th style="text-align: left;">Left Aligned</th>
      <th style="text-align: center;">Center Aligned</th>
      <th style="text-align: right;">Right Aligned</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Left 1</td>
      <td style="text-align: center;">Center 1</td>
      <td style="text-align: right;">Right 1</td>
    </tr>
    <tr>
      <td>Left 2</td>
      <td style="text-align: center;">Center 2</td>
      <td style="text-align: right;">Right 2</td>
    </tr>
  </tbody>
</table>
