# 2. AI-G의 부트 모드
---
AI-G는 다음 부트 모드를 지원합니다: 
- USB 부트 모드 (FWDN)  
- eMMC 부트 모드 (GPIO_SD 그룹)
 
AI-G 보드는 기본적으로 eMMC 부트 모드로 고정되어 있습니다. eMMC 장치에 부트 이미지가 없으면 부트 모드가 자동으로 FWDN 부트 모드로 전환됩니다. 
 
<br/><br/><br/>
 
## 2.1 USB 부트 모드 (FWDN)
--- 
그림 2.1은 부트 모드 스위치와 리셋 스위치를 보여줍니다. USB 부트 모드는 펌웨어 업데이트에 사용됩니다. FWDN 모드로 진입하려면 부트 모드 스위치와 리셋 스위치가 모두 필요합니다. 이미지를 업로드할 때 이 방법을 사용하는 것이 좋습니다.
 
<p align="center"><img src="https://raw.githubusercontent.com/topst-development/Documentation/refs/heads/main/Assets/TOPST%20AI-G/Hardware/2.11%20AI-G%20Boot%20Mode.png" width="550"></p>
<p align="center"><strong>그림 2.1 FWDN 부트 모드 진입을 위한 부트 모드 스위치 및 리셋 스위치 </strong></p><br/>
 
다음 단계는 USB 부트 모드로 진입하기 위한 스위치 조작을 설명합니다. 
1. 전원이 공급되면 리셋 스위치 위의 LED가 켜집니다. 
2. 부트 모드 스위치를 길게 누릅니다. 부트 모드 스위치를 누르면 부트 모드 스위치 위의 빨간색 LED가 켜지는 것을 볼 수 있습니다. LED가 켜진 상태를 유지하도록 스위치를 계속 누르고 계십시오. 
3. 리셋 스위치 LED는 기본적으로 녹색입니다. 리셋 스위치를 한 번 누르면 녹색 LED가 꺼졌다가 다시 켜집니다. 
4. 부트 모드 스위치를 놓습니다. 
 
그림 2.2는 USB 부트 모드(FWDN)로 진입하기 위한 부트 모드 및 리셋 스위치의 작동 순서를 보여줍니다.
