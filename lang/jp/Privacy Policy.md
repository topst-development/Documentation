## TOPST プライバシーポリシー [V.1.0] 2024年1月3日
----

「株式会社テレチップス」（以下「テレチップス」）は、情報主体の自由と権利を保護するために、「個人情報保護法」および関連法令が定める内容を遵守し、適法に個人情報を処理し、安全に管理しています。本プライバシーポリシーは、テレチップスが提供するサービスである「TOPST」の運営に関連して、「個人情報保護法」第30条に基づき、情報主体に個人情報の処理に関する手続きおよび基準を案内し、これに関連する苦情を迅速かつ円滑に処理できるようにするために、以下の通りプライバシーポリシーを策定・公開します。

<br/>

### 1. 個人情報の処理目的

テレチップスは、以下の目的のために個人情報を処理します。処理している個人情報は、以下の目的以外の用途には利用されず、利用目的が変更される場合には、「個人情報保護法」第18条に基づき、別途同意を得るなど必要な措置を履行する予定です。

1. 顧客からの問い合わせに対する回答の提供および状況管理
2. リソース資料の提供
3. イベント参加およびミーティングの問い合わせに対するフィードバックの提供および管理
4. ニュースレター購読のフィードバック提供および管理
5. テレチップスソリューションの発売、イベント情報、ニュースレター形式のEメールマーケティングおよび景品発送への活用

<br/>

### 2. 処理する個人情報の項目

テレチップスは、サービス提供のために必要最小限の範囲で個人情報を収集・利用します。

1) 情報主体の同意を得ずに処理する個人情報項目
情報主体の同意なしに処理する個人情報項目はありません。
 
2) 情報主体の同意を得て処理する個人情報項目
テレチップスは、以下の個人情報項目を「個人情報保護法」第15条第1項第1号および第22条第1項第7号に基づき、情報主体の同意を得て処理しています。

    (1) 顧客からの問い合わせに対する回答の提供および状況管理
      - 収集・利用項目: 名前、Eメール、連絡先、会社名

    (2) リソース資料の提供
    - 収集・利用項目: 名前、Eメール、連絡先、会社名

    (3) イベント参加およびミーティングの問い合わせに対するフィードバックの提供および管理
    - 収集・利用項目: 名前、Eメール、連絡先、会社名

    (4) テレチップスソリューションの発売、イベント情報、ニュースレター形式のEメールマーケティングおよび景品発送への活用
    - 収集・利用項目: 名前、Eメール、連絡先、会社名

3) また、サービス利用過程で以下のような情報が自動的に生成され、収集される場合があります。

- サービス利用記録 : アクセスログ、アクセスIP情報、サービス利用記録
- PC環境情報 : OSバージョン、ブラウザバージョン

モバイル機器利用時 : 端末識別番号、端末OS情報

<br/>

### 3. 個人情報の処理および保有期間

テレチップスは、法令に基づく個人情報の保有・利用期間、または情報主体から個人情報を収集する際に同意を得た個人情報の保有・利用期間内で個人情報を処理・保有します。
 
それぞれの個人情報の処理および保有期間は以下の通りです。

1) 顧客からの問い合わせに対する回答の提供および状況管理 : 同意撤回時まで

2) リソース資料の提供 : 同意撤回時まで

3) イベント参加およびミーティングの問い合わせに対するフィードバックの提供および管理 : 同意撤回時まで

4) テレチップスソリューションの発売、イベント情報、ニュースレター形式のEメールマーケティングおよび景品発送への活用: 同意撤回時まで

テレチップスは、関係法令違反による捜査・調査などが進行中の場合には、当該捜査・調査終了時まで、当該法令に基づき一定期間情報を保有します。

<br/>

### 4. 満14歳未満の児童の個人情報処理

利用者は、自身の個人情報処理に関して、以下のような権利を持つことができます。

- 個人情報の閲覧（照会）を要求する権利

- 個人情報の訂正を要求する権利

- 個人情報の処理停止を要求する権利

- 個人情報の削除要求および同意撤回を要求する権利
- 14歳未満の児童の個人情報を処理する場合には、法定代理人の同意を得る必要があります。

- 法定代理人は、児童の個人情報を照会したり、修正および削除、処理停止および同意撤回などの権利を行使することができます。

- 法定代理人の同意を得るために、児童に法定代理人の氏名、連絡先などの最小限の情報を要求することができ、以下の方法で法定代理人の同意を確認します。
  - 法定代理人の携帯電話本人認証を通じて本人確認を行う方法
  - 法定代理人に同意内容が記載された書面を提供し、署名捺印後に提出させる方法
  - その他、上記に準ずる方法で法定代理人に同意内容を通知し、同意の有無を確認する方法

 
<br/>

### 5. 個人情報の破棄手続きおよび方法

テレチップスは、個人情報の保有期間の経過、処理目的の達成など、個人情報が不要になった場合には、当該情報を遅滞なく破棄します。
 
個人情報の破棄手続きおよび方法は以下の通りです。

1) 破棄手続き
    - テレチップスは、破棄事由が発生した個人情報を選定し、ソーシャルメディアサービス個人情報保護責任者の承認を得て個人情報を破棄します。

1) 破棄方法
    - 電子ファイル形式で保存された個人情報は、記録を再生できない技術的方法を使用して削除されます。紙に出力された個人情報は、シュレッダーで粉砕するか、焼却を通じて破棄します。

 
<br/>

### 6. 個人情報の第三者提供および処理委託
1) テレチップスは、情報主体の個人情報を個人情報の処理目的で明示した範囲内でのみ処理し、情報主体の同意、法律の特別な規定など「個人情報保護法」第17条および第18条に該当する場合にのみ個人情報を第三者に提供し、それ以外には情報主体の個人情報を第三者に提供しません。
 
2) テレチップスは、災害、感染症、急迫した生命・身体の危険を招く事件・事故、急迫した財産損失などの緊急事態が発生した場合、情報主体の同意なしに関係機関に個人情報を提供することができます。

<br/>

### 7. 個人情報の委託処理

テレチップスは、個人情報を第三者に個人情報処理業務を委託しません。

 
<br/>

### 8. 個人情報の国外移転

テレチップスは、情報主体の個人情報を国外に移転または提供しません。

<br/>

### 9. 個人情報の安全性確保措置

テレチップスは、以下のように安全性確保に必要な技術的、管理的、物理的措置を講じています。
 
1) 管理的措置 : 内部管理計画の策定・施行、専任組織の運営、定期的な従業員教育
2) 技術的措置 : 個人情報処理システムなどのアクセス権限管理、アクセス制御システムの設置、個人情報の暗号化、セキュリティプログラムの設置および更新
3) 物理的措置 : 計算機室、資料保管室などのアクセス制御

<br/>

### 10. 個人情報自動収集装置の設置・運営およびその拒否

テレチップスは、ウェブサイトの使用を支援し、関連サービスを提供するためにクッキー(cookie)を運営します。クッキーは、ウェブサイトを運営するサーバーが利用者のブラウザに送信する小さなテキストファイルで、情報主体のコンピュータに保存されます。情報主体はクッキーの保存を拒否することができますが、この場合、ウェブサイトの使用が不便になったり、一部のサービスの場合、利用が困難になることがあります。

▶ クッキー設定方法の例

Internet Explorerの場合 : ウェブブラウザのツールメニュー > インターネットオプション > プライバシー > 設定

Chromeの場合 : ウェブブラウザの設定メニュー > 画面下部の詳細設定を表示 > プライバシーのコンテンツ設定ボタン > クッキー

<br/>

### 11. 情報主体の権利とその行使方法

情報主体は、テレチップスに対していつでも個人情報の閲覧・訂正・削除・処理停止および撤回要求などの権利を行使（以下「権利行使」という）することができます。
権利行使は、テレチップスに対して「個人情報保護法」施行令第41条第1項に基づき、書面、電子メール、ファックス(FAX)などを通じて行うことができ、テレチップスはこれに対して遅滞なく措置を講じます。
- 情報主体は、いつでもサービス担当部署にメールを通じて個人情報の照会・修正・削除を要請することができます。
 
権利行使は、情報主体の法定代理人や委任を受けた者など代理人を通じて行うこともできます。この場合、「個人情報処理方法に関する告示」別紙第11号書式による委任状を提出する必要があります。
情報主体が個人情報の閲覧および処理停止を要求する権利は、「個人情報保護法」第35条第4項および第37条第2項により制限される場合があります。
 
他の法令でその個人情報が収集対象として明示されている場合には、当該個人情報の削除を要求することはできません。
 
テレチップスは、権利行使をした者が本人であるか、正当な代理人であるかを確認します。
 
テレチップスは、権利行使を以下の部署に行うことができます。テレチップスは、情報主体の権利行使が迅速に処理されるよう努めます。
