---

# 2. はじめに
---
Telechips Model Zooが提供するニューラルネットワークモデルをAI-Gボードで実行するには、以下の手順に従ってください。
<br/><br/><br/>

## 2.1 リポジトリのクローン
---
```
$ git clone https://github.com/telechips-ai/TCC750x-Model-Zoo.git

$ cd TCC750x-Model-Zoo
```
<br/><br/><br/>

## 2.2 目的のモデルをAI-Gにコピー
---
モデルフォルダ全体 (yolov5m) をAI-Gにコピーします。
フォルダをそのままコピーするか、以下の例に示すように、以前にリリースされたニューラルネットワークフォルダで使用されていた形式に合わせて yolov5m_quantized に名前を変更できます。
各フォルダには、必要な出力ファイル (.so、.binなど) が含まれています。

```
$ scp -r [network_output_folder] root@192.168.0.100:/path/to/target/
```
**network_output_folder** を実際のフォルダ (例: yolov5m/) に置き換えてください。

yolov5モデルを選択できます。
```
$ cd Object_Detection/YOLO/yolov5/
$ ls 
README.md  yolov5l  yolov5m  yolov5n  yolov5s  yolov5x
```
次に実行します:
```
$ scp -r yolov5m/ root@192.168.0.100:/home/root/
```
<br/><br/><br/>

## 2.3 tcnputestappまたはtcnnappを使用したモデルの実行:
---
次に、ターゲットボード (AI-G) で以下のコマンドを実行します。
```
$ tcnputestapp -d 0 1 -n [network_output_folder_path]
```
または
```
$ tcnnapp -n [network_output_folder_path]
```
